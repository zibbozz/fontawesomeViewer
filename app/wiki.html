Sizing Icons

To change the size of an icon you can simply change the font-size. The icon is handled as an character so if you change the color attribute the icon will change its color. There are also some preset sizes for the icons delivered by the fontawesome framework. The prefixes are fa-xs, fa-sm, fa-lg and fa-2x-fa-10x

Fixed Width Icons

Sometimes a fixed width is very important because without the fa-fw class the icons are not all the same width. This may look very ugly so thankfully this class was added to the framework. In the following examples the icons will have a visible background to see how the classes are working.

With fa-fw

Animating Icons

The fontawesome framework offers two quick ways to animate an icon. The needed classes are fa-spin and fa-pulse. But be careful: some browsers have problems to find the center of the icon so they start to wobble. There are two fixes for this that may work.

  • Switch to the SVG with JavaScript version
  • Set the display of the icon you want to animate to display: block;

Power Transforms

Warning

This feature only works if you use the SVG with JavaScript version!

The following options are going in a new attribute called data-fa-transform. This is one reason why SVG with JS is needed for these options.

Scaling

You can use grow-# and shrink-# to scale icons. You can even use decimals.

shrink-8 shrink-3.35 Original grow-8
The code for the shrink-8 example is
<i class="fas fa-atom fa-4x" data-fa-transform="shrink-8" style="background-color:#DDAAAA"></i>
so you can see how to use the new attribute.

Positioning

This works just like scaling but here you have the options up-#, down-#, left-# and right-#. Keep in mind that you can combine them too!

All Icons are scaled down with shrink-8 so you can see the background for better understanding.

Original up-4 down-4 left-4 right-4

Rotating and Flipping

Instead of using the old method to rotate icons you can now use rotate-#, flip-v and flip-h. This method gives you the freedom to rotate the icon by any degree.

Original rotate-35 rotate-74.5 flip-h

Masking Icons

Warning

This feature only works if you use the SVG with JavaScript version!

It is also possible to mask icons instead of simply stacking them. This way anything behind the icons will show through the icons.

Advanced example

The code looks like this

<i class="fab fa-facebook-f fa-4x" data-fa-transform="shrink-4 down-2 right-1.5" data-fa-mask="fas fa-circle" style="background-color:#DDAAAA"></i>

The main icon is the icon you want to cut out. The data-fa-mask icon is the one that will get cut. All data-fa-transform options will only apply to the main icon.

Layering, Text and Counters

Layers are an alternative to stacking icons. With layers the background wont show through too. Layers are a little bit more advanced because you can use text and counters too.

Examples

15 164

Code

<div class="fa-4x">

    <span class="fa-layers fa-fw" style="background-color:#DDAAAA">
        <i class="fas fa-circle" style="color:#FF4422"></i>
        <i class="fas fa-times fa-inverse" data-fa-transform="shrink-5"></i>
    </span>

    <span class="fa-layers fa-fw" style="background-color:#DDAAAA">
        <i class="fas fa-play" data-fa-transform="rotate--90 grow-2"></i>
        <i class="fas fa-sun fa-inverse" data-fa-transform="shrink-12 up-4"></i>
        <i class="fas fa-moon fa-inverse" data-fa-transform="shrink-12 down-4 left-4"></i>
        <i class="fas fa-star fa-inverse" data-fa-transform="shrink-12 down-4 right-4"></i>
    </span>

    <span class="fa-layers fa-fw" style="background-color:#DDAAAA">
        <i class="fas fa-calendar" data-fa-transform="shrink-1"></i>
        <span class="fa-layers-text fa-inverse" data-fa-transform="shrink-7 down-2.5">15</span>
    </span>

    <span class="fa-layers fa-fw" style="background-color:#DDAAAA">
        <i class="fas fa-envelope"></i>
        <span class="fa-layers-counter" style="background-color:#FF4422">164</span>
    </span>

</div>

You can also change the position of the counter with fa-layers-bottom-left, fa-layers-bottom-right, fa-layers-top-left and fa-layers-top-right. You need to assign one of these classes to the span that contains the number.
